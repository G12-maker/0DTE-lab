# Example Config Guide

Use the example that matches your situation.

## `demo_no_theta.yaml`

Use this if:

- you do not have Theta
- you just want one working demo
- you want the easiest first run

This example uses bundled sample data and the `massive_simplified` path.

## `theta_smoke.yaml`

Use this if:

- you already have local Theta data
- you want to validate your local paths
- you want a shorter, safer first Theta run

This example is meant for a smoke test, not a full historical run.

## `opening_momentum_base.yaml`

Use this if:

- you want a fuller Theta example
- you want a starting point for your own edits
- you want to study the current default opening momentum setup

This is the main editable template.
