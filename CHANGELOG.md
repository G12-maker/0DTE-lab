# Changelog

## 0.1.0

Initial public alpha for `QQQ 0DTE` single-leg research.

Included in this version:

- standalone `odte-lab` CLI
- `QQQ 0DTE` backtest and replay entrypoints
- `theta_realistic` and `massive_simplified` data modes
- bundled Massive sample data for onboarding
- standardized outputs:
  - `base_trades.csv`
  - `trades.csv`
  - `summary.csv`
  - `run_config.json`
  - `coverage.json`
- basic tests and example configs

Known limitations:

- alpha maturity
- no multi-leg support
- no broker integration
- no built-in chart UI
